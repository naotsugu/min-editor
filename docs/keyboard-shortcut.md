# Keyboard Shortcut

| shortcut     | description                      |
|--------------|----------------------------------|
| `Ctrl` + `↑` | Half-page up                     |
| `Ctrl` + `↓` | Half-page down                   |
| `Ctrl` + `→` | Move the caret to the right word |
| `Ctrl` + `←` | Move the caret to the left word  |
| `Ctrl` + `p` | Open the command pallet          |
| `Ctrl` + `l` | Toggle line wrap                 |
| `Ctrl` + `s` | Save current file                |
