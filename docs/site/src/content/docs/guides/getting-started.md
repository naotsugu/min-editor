## Installing

Download the latest min-editor release and unzip.
You can use the application by launching the executable file.

### Launching app on macOS

- [min-editor-mac.zip](https://github.com/naotsugu/min-editor/releases/latest/download/min-editor-mac.zip)
- [min-editor-mac-aarch64.zip](https://github.com/naotsugu/min-editor/releases/latest/download/min-editor-mac-aarch64.zip)

### Launching app on Windows

- [min-editor-win.zip](https://github.com/naotsugu/min-editor/releases/latest/download/min-editor-win.zip)

### Launching app on Linux

- [min-editor-linux.zip](https://github.com/naotsugu/min-editor/releases/latest/download/min-editor-linux.zip)


## Uninstalling

Delete executable files and configuration files.
